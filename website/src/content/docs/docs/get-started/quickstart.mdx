SlateDB is implemented in Rust. The repository also ships official Go, Python, and Java bindings, but this quick start covers the Rust API directly.

## Installation

Add the following dependencies to your project:

```bash
cargo add slatedb tokio --features tokio/macros,tokio/rt-multi-thread
```

## Usage

SlateDB uses [`tokio`](https://crates.io/crates/tokio) as its async runtime and [`object_store`](https://docs.rs/object_store/latest/object_store/) to interact with object storage. The code below shows a simple in-memory example.
