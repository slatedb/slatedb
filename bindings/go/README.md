# SlateDB Go Binding

`bindings/go` contains the Go module for SlateDB generated from the UniFFI surface in
`bindings/uniffi`.

The published module path is:

```text
slatedb.io/slatedb-go
```

## Status

This binding is generated code with a very small amount of repository glue:

- `uniffi.go` and `uniffi.h` are generated by `uniffi-bindgen-go`
- `cgo_flags.go`, `go.mod`, and this `README.md` are maintained by hand

There is no handwritten Go wrapper layer yet.

## Prerequisites

- Rust toolchain from this repository
- Go 1.25.0 or newer
- CGO enabled
- A working C toolchain
- `uniffi-bindgen-go` `0.5.0+v0.29.5`

Install the generator with:

```bash
cargo install uniffi-bindgen-go --git https://github.com/NordSecurity/uniffi-bindgen-go --tag v0.5.0+v0.29.5
```

## Regenerate

From the repository root:

```bash
./scripts/generate-go-uniffi.sh
```

The script builds `slatedb-uniffi`, runs `uniffi-bindgen-go`, and copies the generated
`uniffi.go` and `uniffi.h` files into `bindings/go`.

## Basic Example

```go
package main

import (
	"bytes"
	"fmt"

	slatedb "slatedb.io/slatedb-go"
)

func main() {
	store, err := slatedb.ObjectStoreResolve("memory:///")
	if err != nil {
		panic(err)
	}
	defer store.Destroy()

	builder := slatedb.NewDbBuilder("/tmp/slatedb-go-cache", store)
	defer builder.Destroy()
	db, err := builder.Build()
	if err != nil {
		panic(err)
	}
	defer db.Destroy()

	if _, err := db.Put([]byte("hello"), []byte("world")); err != nil {
		panic(err)
	}

	value, err := db.Get([]byte("hello"))
	if err != nil {
		panic(err)
	}
	if value == nil || !bytes.Equal(*value, []byte("world")) {
		panic("unexpected value")
	}

	fmt.Println(string(*value))

	if err := db.Shutdown(); err != nil {
		panic(err)
	}
}
```

## Build And Test

From the repository root:

```bash
cargo build -p slatedb-uniffi
./scripts/generate-go-uniffi.sh
```

Then run Go validation from `bindings/go`:

```bash
cd bindings/go
CGO_ENABLED=1 CGO_LDFLAGS="-L$(pwd)/../../target/debug" \
LD_LIBRARY_PATH="$(pwd)/../../target/debug:${LD_LIBRARY_PATH}" \
DYLD_LIBRARY_PATH="$(pwd)/../../target/debug:${DYLD_LIBRARY_PATH}" \
go test ./...
```

## Runtime Linking

The generated Go package links against the shared library produced by `slatedb-uniffi`.
Make sure your runtime loader can find the library from `target/debug` or `target/release`.

Examples:

```bash
export LD_LIBRARY_PATH="$(pwd)/target/debug:$(pwd)/target/release:${LD_LIBRARY_PATH}"
export DYLD_LIBRARY_PATH="$(pwd)/target/debug:$(pwd)/target/release:${DYLD_LIBRARY_PATH}"
```

## Vanity Import Path

The repository directory for this module is `bindings/go`, but consumers should import:

```go
import "slatedb.io/slatedb-go"
```

The imported package identifier is `uniffi`.
